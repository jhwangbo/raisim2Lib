granular_media
==============

.. image:: ../../../image/granular_media.png
   :alt: granular_media example
   :width: 100%

Installed executable: ``granular_media``.

Run the installed executable and connect with ``rayrai_raisim_tcp_viewer``:

.. code-block:: bash

   <raisim-install>/bin/granular_media

On Windows, run ``granular_media.exe`` instead.

This example shows ANYmal standing on a native granular bed with instanced granular-particle visuals.
