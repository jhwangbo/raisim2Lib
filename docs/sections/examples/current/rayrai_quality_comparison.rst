rayrai_quality_comparison
=========================

``rayrai_quality_comparison`` renders comparison images for rayrai quality
presets and writes a small report. It is built from
``visualizer/rayrai/examples/rayrai_quality_comparison.cpp``.

Run:

.. code-block:: bash


   <raisim-install>/bin/rayrai_quality_comparison --out=/tmp/rayrai_quality --frames=4

The output directory contains preset images and ``report.md``.
