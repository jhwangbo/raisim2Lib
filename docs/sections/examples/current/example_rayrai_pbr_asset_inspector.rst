example_rayrai_pbr_asset_inspector
==================================

``example_rayrai_pbr_asset_inspector`` opens a rayrai window containing bundled
glTF PBR sample assets, including helmets, camera, lantern, boombox, avocado,
and water bottle assets. It configures the Ultra render-quality preset, HDR
environment lighting, SSAO, bloom, shadows, and a matte floor.

Run:

.. code-block:: bash


   <raisim-install>/bin/example_rayrai_pbr_asset_inspector
   <raisim-install>/bin/example_rayrai_pbr_asset_inspector --screenshot /tmp/pbr_assets.png

Options include ``--no-shadows``, ``--no-ssao``, ``--no-post``, and
``--screenshot``.
