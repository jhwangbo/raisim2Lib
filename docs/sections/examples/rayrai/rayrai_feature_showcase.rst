rayrai_feature_showcase
=======================

``rayrai_feature_showcase`` is built from ``visualizer/rayrai/examples``. It
renders the current rayrai feature showcase and writes image outputs plus
metrics. It is also used by the ``rayrai_check_showcase`` custom target.

Run directly:

.. code-block:: bash


   <raisim-install>/bin/rayrai_feature_showcase --out=/tmp/rayrai_showcase
