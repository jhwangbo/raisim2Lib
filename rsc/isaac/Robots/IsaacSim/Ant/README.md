# Isaac Sim Ant USD Asset

This directory contains the NVIDIA Isaac Sim Ant USD files used by the
`nvidia_usd_robots` example.

Source:
`https://omniverse-content-production.s3-us-west-2.amazonaws.com/Assets/Isaac/5.1/Isaac/Robots/IsaacSim/Ant/`

Downloaded files:
- `ant.usd`
- `ant_colored.usd`
- `ant_instanceable.usd`
- `configuration/ant_robot_schema.usd`
- `configuration/ant_colored_robot_schema.usd`
- `configuration/ant_instanceable_robot_schema.usd`

NVIDIA's Isaac Sim robot asset documentation lists `IsaacSim/Ant/ant.usd` with
an MIT license. The downloaded asset directory also includes
`ant-LICENSE.txt`.
