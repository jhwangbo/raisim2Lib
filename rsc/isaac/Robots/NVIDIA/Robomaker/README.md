# AWS Robomaker Jetbot USD Asset

This directory contains the NVIDIA Isaac Sim AWS Robomaker Jetbot USD files used
by the `nvidia_usd_robots` example.

Source:
`https://omniverse-content-production.s3-us-west-2.amazonaws.com/Assets/Isaac/5.1/Isaac/Robots/NVIDIA/Robomaker/`

Downloaded files:
- `aws_robomaker_jetbot.usd`
- `configuration/aws_robomaker_jetbot_robot_schema.usd`

NVIDIA's Isaac Sim robot asset documentation lists
`NVIDIA/Robomaker/aws_robomaker_jetbot.usd` with an MIT license. The MIT license
text is included in `LICENSE`.
