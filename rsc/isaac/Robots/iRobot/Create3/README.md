# iRobot Create 3 USD Asset

This directory contains the NVIDIA Isaac Sim iRobot Create 3 USD files used by
the `nvidia_usd_robots` example.

Source:
`https://omniverse-content-production.s3-us-west-2.amazonaws.com/Assets/Isaac/5.1/Isaac/Robots/iRobot/Create3/`

Downloaded files:
- `create_3.usd`
- `configuration/create_3_robot_schema.usd`

NVIDIA's Isaac Sim robot asset documentation lists `iRobot/Create3/create_3.usd`
with a BSD-3 license. The BSD-3 license text is included in `LICENSE`.
