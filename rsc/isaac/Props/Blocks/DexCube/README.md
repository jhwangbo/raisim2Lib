# Textured Cube Visual Asset

This directory contains a small procedural cube visual used by the
`shadow_hand_usd_cube` example.

The dynamic body in the example is still a native RaiSim box. This procedurally
generated OBJ/MTL/PNG asset is only a visual proxy so the TCP viewer shows a
textured cube without redistributing NVIDIA's separate DexCube prop asset.

Files:
- `dex_cube_textured.obj`
- `dex_cube_textured.mtl`
- `dex_cube_texture.png`
